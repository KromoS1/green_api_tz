### Локальный запуск

для того что бы запустить проект локально выполните следующие команды
>`yarn`

>`yarn dev`


либо вы можете выполнить команду сборки и выполнить команды запуска продакшен сборки

>`yarn`

> `yarn build`

>`yarn preview`
